# Changelog

All notable changes to the "antigravity-skill-manager" extension will be documented in this file.

## [1.0.0] - 2026-03-15

### Added
- Initial release of Antigravity Skill Manager.
- Local skill detection for global and workspace directories.
- Remote skill discovery from GitHub repositories.
- One-click installation of remote skills.
- Read-only preview of remote skill files.
- Skill scaffolding from structural templates.
- Management UI for adding/removing remote GitHub repositories.
- Customizable global skills path and invalid skill hiding.
