# DECISIONS.md

# ADR-01: Use TypeScript for Extension Development
**Status**: `ACCEPTED`
**Context**: User requested a VSCode extension and is open to learning new languages.
**Decision**: Use TypeScript as it is the standard and most well-supported language for VSCode extensions, providing the best developer experience and API stability.

# ADR-02: Project Organization (GSD)
**Status**: `ACCEPTED`
**Context**: Implementing the "Get Shit Done" methodology.
**Decision**: Use the `.gsd` directory for all project management, roadmap, and specification documents to maintain a high-leverage execution flow.
